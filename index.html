- this is the main search page

You can search for anything on Twitter right here. From topics to hastags, videos, photos, etc. - just type in the search keyword(s).

You can select the language and the type of tweet you want.

- this is where you get tweet on a user's timeline.

You can get tweets from any twitter user here. Just type in the twitter username and you're good to go.

You won't be able to see tweets from private or suspended accounts.

- this is where your favorite tweets are saved

As a logged in user, you can save and view your favorite tweets later.

You have a limit of 10 collections. There is no limit on the number of tweets you can save to a collection at this time.

*In order not to violate Twitty privacy policy, tweet contents are not cached, only the tweet references are. Hence, if a user deletes a tweet or make their account private, you will no longer have access to that tweet.
